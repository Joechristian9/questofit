About Our Game

We are IT students passionate about creating meaningful and creative digital experiences. Quest of the IT Hero is an educational 2D RPG where players learn programming and IT concepts through interactive missions instead of combat. Players explore a digital world, travel to designated areas, and complete tasks such as answering programming questions, identifying correct syntax, and debugging code. The game transforms learning IT fundamentals into a fun, guided adventure that improves problem-solving and logical thinking skills.

🎯 Core Gameplay
  • No attack or fighting mechanics
  • Mission-based progression system
  • Explore villages, labs, terminals, and servers
  • Programming syntax questions
  • Debugging challenges
  • Logic and IT-related quizzes
🎨 Visual Style
  • Pixel-art / retro 2D RPG aesthetic
  • Calm, intelligent, and adventurous atmosphere
  • Focus on learning and exploration
